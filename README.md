# Deep Learning Projekt – Ansiktsuttryck

Detta projekt använder en CNN-modell för att klassificera ansiktsuttryck från bilder.

Projektet innehåller:
- träning av CNN-modeller
- jämförelse mellan två modeller
- analys av accuracy och loss
- prediktion på ny data
- sparad tränad modell

Dataset:
FER-2013
